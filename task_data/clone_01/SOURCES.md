## Included Sources

### Silver

1. Yanisch-Perron C, Vieira J, Messing J. *Improved M13 phage cloning vectors and host strains: nucleotide sequences of the M13mp18 and pUC19 vectors.* Gene. 1985. DOI: https://doi.org/10.1016/0378-1119(85)90120-9
   - Used for: the canonical pUC19 sequence and the 2686 bp vector length.

2. Weiss B, Richardson CC. *Enzymatic breakage and joining of deoxyribonucleic acid. I. Repair of single-strand breaks in DNA by an enzyme system from Escherichia coli infected with T4 bacteriophage.* PNAS. 1967. DOI: https://doi.org/10.1073/pnas.57.4.1021
   - Used for: foundational description of T4 DNA ligase activity, anchoring the "T4 DNA ligase for cohesive ends" decision.

3. Vieira J, Messing J. *The pUC plasmids, an M13mp7-derived system for insertion mutagenesis and sequencing with synthetic universal primers.* Gene. 1982. DOI: https://doi.org/10.1016/0378-1119(82)90015-4
   - Used for: blue-white screening semantics reused for Clone-01's post-transformation colony PCR verification.

### Bronze

1. NEB. *Double-Digest Finder (EcoRI + BamHI).* https://www.neb.com/en-us/tools-and-resources/selection-charts/double-digest-finder
   - Used for: buffer compatibility and enzyme-pair confirmation.

2. NEB. *Optimizing Restriction Endonuclease Reactions.* https://www.neb.com/en-us/protocols/2012/12/07/optimizing-restriction-endonuclease-reactions
   - Used for: the 60-minute minimum digest duration.

3. NEB. *Heat Inactivation Reference.* https://www.neb.com/en-us/tools-and-resources/selection-charts/heat-inactivation
   - Used for: the 65 C heat-inactivation condition for EcoRI and BamHI.

4. NEB. *DNA Ligation with T4 DNA Ligase (M0202).* https://www.neb.com/en-us/protocols/0001/01/01/dna-ligation-with-t4-dna-ligase-m0202
   - Used for: canonical vector:insert molar ratio (1:3) and standard 16 C overnight ligation.

5. NEB. *Quick Ligation Protocol (M2200).* https://www.neb.com/en-us/protocols/0001/01/01/dna-ligation-with-t4-dna-ligase-m2200
   - Used for: 10-minute room-temperature T4 ligation as an acceptable alternative.
